# DemonMarket
#### 一个系统商店插件，玩家可以出售物品。采用调优后的幂指函数与反函数混合收敛物价。用于解决工业类服务器某些物品产能过高导致服务器经济失衡的问题。
## 使用方法
#### 使用用户命令/demonmarket 命令简写为/mt
+ /mt sell 卖掉手里的东西
+ /mt sell all 卖掉背包里和手中同样的所有东西
